# notate

Compiles guitar tabs annotated with piano chords to PDF.

## Installation

Requires Lilypond and Pandoc.

```bash
git clone https://github.com/euugenechou/notate.git
cargo install --path ./notate
```

## Usage

See program help for usage.

```bash
notate --help
```
